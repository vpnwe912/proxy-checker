# Assets

Эта папка содержит исходные ресурсы для приложения.

## Файлы

- **appicon.png** — иконка приложения (используется в README и для генерации .ico)
- **screenshot.png** — скриншот приложения для README (нужно добавить)

## Как добавить скриншот

1. Запустите приложение:
   ```powershell
   .\build\bin\ProxyChecker.exe
   ```

2. Загрузите несколько прокси и проверьте их

3. Сделайте скриншот окна приложения (Win + Shift + S)

4. Сохраните скриншот как `screenshot.png` в эту папку

5. Раскомментируйте строку в README.md:
   ```markdown
   ![Proxy Checker Screenshot](assets-src/screenshot.png)
   ```

## Рекомендации для скриншота

- Размер: 1200x800 или больше
- Формат: PNG
- Покажите основные функции:
  - Список прокси с результатами проверки
  - Панель настроек слева
  - Статус монитора и 3proxy
  - Несколько рабочих (зеленых) прокси
